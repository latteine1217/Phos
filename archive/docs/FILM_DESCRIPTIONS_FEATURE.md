# 底片描述卡片功能

## 📋 新功能說明

在側邊欄的**底片選擇框下方**，現在會動態顯示當前選中底片的詳細資訊卡片。

---

## 🎨 卡片內容

每張底片卡片包含以下資訊：

### 1. 標題區
- **底片名稱**（大標題，紅色強調）
- **品牌** · **類型**（彩色/黑白） · **ISO 值**

### 2. 描述區
- 一句話核心描述
- 突出底片的主要特性與定位

### 3. 特點標籤
- 3個特點標籤（紅色底色）
- 快速了解技術特性

### 4. 適用場景
- 推薦的拍攝題材
- 幫助選片決策

## 🎯 使用方式

1. 啟動 Streamlit 應用：
   ```bash
   streamlit run Phos_0.3.0.py
   ```

2. 在左側邊欄選擇底片

3. 選擇框下方會**即時顯示**該底片的詳細資訊卡片

4. 根據卡片資訊決定是否使用該底片

- **漸層背景**: 深色半透明，融入整體UI

- **標籤樣式**: 紅色底色，圓角設計
- **響應式**: 自動適應側邊欄寬度
